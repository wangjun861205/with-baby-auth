## Singup

- URL: /signup
- Method: POST
- Request Headers:

  | Name         | Value            | Is Required |
  | ------------ | ---------------- | ----------- |
  | Content-Type | application/json | 是          |

- Request Body Structure:

  - Example:

    ```json
    {
      "username": "test",
      "password": "test"
    }
    ```

  - Field Description:

  | Name     | Type   | Is Required | Description      |
  | -------- | ------ | ----------- | ---------------- |
  | username | string | 是          | 所要注册的用户名 |
  | password | string | 是          | 密码             |
